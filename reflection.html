Responsive Designing:-- Responsive design is GUI that means graphic user interface design strategy for creating content that adapts to different screen sizes without losing its quality. Responsive design ensures usability and satisfaction as, it is an approach that tries to make web pages render well on a variety of devices and window or screen sizes from the smallest to the largest display size. By applying methods, responsive design changes the layout of a web page to the viewing environment. There several techniques used in responsive designing. It makes coding of web applications quite easy and reliable. Also, it saves the time of programmer as he need to code only one time for all the devices and web browsers.

Responsive Design in Web Technologies:- Responsive designing is all about display of web pages that is to make them look good. Responsive designing highly influences web technology. Nowadays, responsive designing is highly used by the web developers while making web applications. The use of HTML (Hyper-Text markup language) and CSS (Cascading Style Sheet) to automatically resize, hide, minimize, or enlarge a website to make it seem excellent on all devices like desktops, phones, and etc is known as responsive web designing. There are several things that are done in responsive web design like setting the viewport, setting images to nicely fit in any type of browser like google chrome or Firefox, adjusting size of the text and many more things are done to make appropriate and good-looking web page or website. This is how all the websites that we open in our laptop and mobile conveniently adjust in both screen that big screen of laptop and small screen of phones. In other words, responsive design is responsible for creating convenient web applications for small screen and big screen devices. This proves that responsive designing plays a vital role in web technology. It is so because a good looking and nicely arranged web pages or websites are easier to read or look any other stuff in it. Moreover, it attracts users because users prefer well-arranged, attractive, and appealing web applications.
